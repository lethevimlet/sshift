Terminal Reattachment Test

Test Instructions

This test verifies that terminals are properly reattached when layouts change:

  1. Open the main application in a new tab
  2. Create an SSH connection
  3. Change the layout (e.g., to 2-panel horizontal)
  4. Verify the terminal content is still visible and functional
  5. Change back to single layout
  6. Verify the terminal is still working

Note: Open the browser console to see detailed logs.
