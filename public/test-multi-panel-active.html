Multi-Panel Active Terminals Test

Test Description

This test verifies that each panel in a multi-panel layout can have its own active terminal.

When switching from a single panel layout to a multi-panel layout, each visible panel should display an active terminal.

Manual Test Steps

  1. Open the SSHIFT application
  2. Create 2-3 SSH tabs in single panel layout
  3. Switch to 2-column layout
  4. Verify: Each panel should show one active terminal (visible terminal-wrapper with 'active' class)
  5. Click on a tab in the left panel - only that panel's terminal should change
  6. Click on a tab in the right panel - only that panel's terminal should change
  7. Both panels should maintain their own active terminal independently
