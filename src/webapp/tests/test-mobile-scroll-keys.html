💻 Implementation Details

Visual Viewport API

if (window.visualViewport) { const updatePosition = () => { const keyboardHeight = window.innerHeight - window.visualViewport.height; if (keyboardHeight > 0) { mobileKeysBar.style.bottom = `${keyboardHeight}px`; } }; window.visualViewport.addEventListener('resize', updatePosition); }

Prevent Keyboard Collapse

key.addEventListener('mousedown', (e) => { e.preventDefault(); }); key.addEventListener('touchstart', (e) => { e.preventDefault(); }, { passive: false });

Auto-scroll Logic

// Only auto-scroll if user is at bottom if (session.isAtBottom) { terminal.scrollToBottom(); }
